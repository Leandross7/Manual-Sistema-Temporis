1. Introdução

Bem-vindo ao Sistema Temporis, a solução completa para a gestão de frequência, banco de horas e compensações da Secretaria Municipal de Cidadania e Assistência Social (SMCAS) de Capão do Leão.

Este manual tem como objetivo ser um guia definitivo para todos os usuários, detalhando as funcionalidades disponíveis em cada um dos três ambientes do sistema e para cada perfil de acesso.

1.1. Os Três Ambientes do Sistema

O Temporis é dividido em três portais distintos, cada um com um propósito específico:

  1. Sistema Temporis (Painel de Gestão): O ambiente principal para a administração e gestão da frequência dos servidores. Acessado por Validadores e Operadores.
  2. Portal do Servidor: Uma área de consulta para que cada servidor possa visualizar a sua própria frequência e extrato de horas de forma transparente e autónoma.
  3. Painel Super Admin: Uma interface de controlo de alto nível, destinada à administração dos perfis de acesso ao sistema de gestão.

2. Visão Geral dos Perfis de Acesso

O sistema possui quatro tipos de usuários, cada um com um nível de permissão específico:

Servidor

Acede apenas ao Portal do Servidor para consulta dos seus próprios dados. Não possui acesso ao sistema de gestão.

Operador

Um perfil de gestão operacional. Pode visualizar os dados de todos os servidores, lançar, editar e salvar as frequências que ainda não foram validadas.

Validador

O perfil de gestão completo. Tem acesso total ao Sistema Temporis, incluindo todas as permissões do Operador, mais a capacidade de cadastrar/editar servidores, validar frequências (travando-as para edição), gerir afastamentos e aceder a todos os relatórios.

Super Admin

O administrador do sistema. Acede ao Painel Super Admin para gerir os perfis dos Operadores e Validadores.

3. Ambiente 1: Sistema Temporis (Painel de Gestão)

Este é o coração operacional do sistema, onde a gestão da efetividade acontece.

3.1. Acesso ao Sistema

  • URL: https://controle-de-efetividade-smcas.web.app
  • Login: Utilize o e-mail e a senha cadastrados. Apenas usuários com perfil Validador ou Operador podem aceder a este ambiente.

3.2. Painel Principal (Dashboard)

Ao fazer o login, verá o "Painel de Servidores", que lista todos os servidores cadastrados. A partir daqui, pode iniciar todas as ações de gestão.

3.3. Funcionalidades do Perfil "Validador"

O Validador tem controlo total sobre as operações de gestão.

3.3.1. Cadastrar e Editar Servidores

  • Para Adicionar: Clique no botão "Adicionar Servidor". Preencha os dados cadastrais, com atenção especial à "Carga Horária Semanal", que é a base para todos os cálculos de horas.
  • Para Editar: Num cartão de servidor, clique no ícone de lápis para abrir o formulário e atualizar as informações.

3.3.2. Lançamento e Gestão da Frequência

Esta é a tela principal de trabalho, partilhada com o Operador.

  1. Acesso: No cartão do servidor desejado, clique em "Ver Frequência".
  2. Lançamento de Horas: Preencha os campos "Entrada" e "Saída" para cada dia trabalhado.
  3. Gestão de Folgas: Para registar uma folga a ser compensada, selecione o status "Folga a Compensar".
  4. Lançamento de Horas Externas: Utilize os campos "HE EXT." para registar horas trabalhadas fora do local.
  5. Desbloqueio de Dias: Para lançar horas em fins de semana ou feriados, clique no botão "Desbloquear".

3.3.3. A Lógica da Conciliação e a Validação (Poder do Validador)

Ao clicar no botão "Salvar Frequência", o sistema executa a conciliação automática.

  • Validação: Após salvar e rever os dados, o Validador deve ir à tela de Relatório Mensal e clicar no botão "Validar" para a frequência do servidor.
  • Importante: Ao ser validada, a frequência é travada e não pode mais ser editada por um Operador, garantindo a integridade dos dados para o fecho do mês. O Validador pode, se necessário, reverter o status para "Pendente" para permitir novas edições.

3.3.4. Gestão de Afastamentos

  • No painel principal, no cartão do servidor, clique em "Gerir Afastamentos".
  • Selecione o tipo (Férias ou Licença Saúde) e preencha as datas.

3.3.5. Relatórios e Logs

O Validador tem acesso exclusivo aos botões no topo do painel para:

  • Relatório Mensal: Gera um PDF consolidado da efetividade de todos os servidores.
  • Banco de Horas: Permite gerar um extrato detalhado da movimentação dos bancos de horas.
  • Log de Atividades: Exibe um histórico de todas as ações importantes realizadas no sistema.

3.4. Funcionalidades do Perfil "Operador" (Revisado e Corrigido)

O Operador é o perfil de linha de frente para a manutenção diária da frequência.

  • Acesso Total à Frequência: Pode visualizar a lista de todos os servidores e aceder à tela de "Lançamento de Frequência" de cada um.
  • Lançamento e Edição: O Operador pode e deve preencher as horas de entrada, saída, horas externas, status ("Folga a Compensar") e salvar as alterações clicando em "Salvar Frequência".
  • Limitação: A permissão de edição do Operador encerra-se quando um Validador altera o status da frequência para "Validado". A partir desse momento, o Operador só pode visualizar os dados daquele mês.
  • Restrições: O Operador não pode cadastrar ou editar servidores, gerir afastamentos, validar frequências ou aceder à área de Relatórios e Logs.

4. Ambiente 2: Portal do Servidor

Um ambiente de consulta projetado para transparência e autonomia.

4.1. Acesso ao Portal

  • URL: https://controle-de-efetividade-smcas.web.app/portal
  • Login: O servidor deve utilizar o seu CPF (com pontos e hífen) e a sua Data de Nascimento.

4.2. Funcionalidades

O portal é simples e direto:

  • Minha Frequência: Permite ao servidor visualizar a sua folha de ponto.
  • Meu Banco de Horas: Apresenta um extrato completo, mês a mês, dos saldos de horas.

5. Ambiente 3: Painel Super Admin

A área de controlo de acesso para os gestores do sistema.

5.1. Acesso ao Painel

  • URL: https://controle-de-efetividade-smcas.web.app/admin
  • Login: Utilize o e-mail e a senha específicos do Super Administrador.

5.2. Funcionalidades

  • Gerir Usuários: Altera os perfis entre "Validador", "Operador" e "Servidor".
  • Criar Acesso ao Portal: Permite que um gestor também aceda ao Portal do Servidor.
  • Logs de Auditoria: Exibe um registo de ações críticas.

Este manual reflete o estado do Sistema Temporis na versão consolidada de produção.
